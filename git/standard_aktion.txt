Drei Schritte zum Aktualisieren:	

1. Sicherstellen, dass Sie in der Basis des verwalteten																			Verzeichnisses sind ( /home/pan/Dokumente/AuK_fit4h) hier liegt der Ordner Git

2. git add . -A
   #  . steht für das aktuelle Verzeichnis
   # -A steht für alle geänderten, hinzugefügten, gelöschten Dateien

3. git commit -am 'Was passiert ist' muss immer was reingeschrieben werden
	# Bestätigung der Änderungen lokal

4. git push origin master
	# Hochladen der neuen Version auf Server
